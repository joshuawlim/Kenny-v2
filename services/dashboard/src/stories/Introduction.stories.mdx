## ðŸŽ¨ Design System

### Color Palette
- **Primary Green**: `#14b88a` - Used for active states and success indicators
- **Dark Green**: `#0e3b2e` - Primary brand color
- **Background**: `#081f1a` - Main background
- **Surface**: `rgba(255, 255, 255, 0.06)` - Card backgrounds with transparency
- **Border**: `rgba(255, 255, 255, 0.12)` - Subtle borders
- **Text**: `rgba(255, 255, 255, 0.95)` - Primary text

### Typography
- **Font Family**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **Headings**: Weight 600 for emphasis
- **Body**: Weight 400 for readability

### Visual Effects
- **Glass Morphism**: Cards use `backdrop-filter: blur(20px)` for a frosted glass effect
- **Gradients**: Radial gradients create depth and visual interest
- **Smooth Transitions**: 0.2s ease transitions for interactive elements
